1. Что будет исключаться в ./Homework/13-Terraform/.gitignore:
все файлы .tfvars, которые могут содержать конфиденциальные данные, например
пароль, секретные ключи и другие секреты. Они не должны быть частью версии
контроль, поскольку они представляют собой точки данных, которые потенциально конфиденциальны и подвержены меняться в зависимости от окружающей среды.
